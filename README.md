# Github-Page
Criando site do meu portfólio.
Nesta área será desenvolvido o meu portfólio pessoal e profissional, referente a entrega intermediária da diciplina de Bootcamp I.
## Sobre
Este repositório foi desenvolvido como parte da atividade de Portfólio Digital com Versionamento, tendo como objetivo organizar projetos acadêmicos, documentação profissional e materiais desenvolvidos durante a graduação em Engenharia de Software.

O projeto utiliza Git e GitHub para versionamento, além de integrar ferramentas como Google Sites, Canva e YouTube para apresentação do portfólio.
